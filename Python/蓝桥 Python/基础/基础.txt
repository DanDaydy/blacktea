循环控制

for ... in ...            for循环
	点到为止，设定次数，次数够为止

while ...                while循环
	不达目的誓不罢休，设定条件，条件满足为止

for 循环 适用于 已知循环次数 的循环，所以后面跟的是次数或区间，到达指定次数就停止
while 后面跟的是一个条件，只要条件满足，这个循环就会一直进行下去

--------------------------

break 和 continue

break 表示停止当前循环
continue 表示跳过当前循环轮次，去执行下一轮循环
